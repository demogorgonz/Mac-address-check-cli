# Mac-address-check-cli
Check mac addresses for vendors from CLI

Requirements:
php-cli
or
php5-cli

## Usage:
/usr/bin/php mac.php MAC

## Example
/usr/bin/php mac.php FC:FB:FB:01:FA:21

Script is utilizing macvendors.com API.
